# ConvNet v0.1.0

Initial public release of the from-scratch C++ CNN project.

## Highlights

- Stage 1 inference engine with `Tensor`, `Conv2D`, `ReLU`, `MaxPool2D`, `Flatten`, `Linear`, and `Softmax`
- Stage 2 training support with backward propagation, `CrossEntropyLoss`, and `SGD`
- Native MNIST IDX loading
- Config-driven model construction
- Saved model artifacts with manifest-based loading
- Checkpoint resume support
- Standalone evaluation executable
- Numerical gradient checking for trainable layers
- CI via GitHub Actions
- Repeat-based benchmark tooling

## Included Executables

- `cnn_infer`
- `cnn_first_milestone`
- `cnn_train_demo`
- `cnn_mnist_train`
- `cnn_eval`
- `cnn_benchmark`

## Representative Result

Using the current MNIST config:

- `10,000` training samples
- `10` epochs
- final test accuracy: `0.9464`

## Notes

- Small one-epoch runs are intended as lifecycle smoke tests, not model-quality benchmarks.
- Model architecture configs and saved model artifacts are separate concepts in this release.
